NativeBASS Plugin

This plugin is based on the BASS sound library (see www.un4seen.com). To be more exact, it uses a Java port of this library, called NativeBASS (http://jerome.jouvie.free.fr/nativebass/index.php). It uses the mp3 decoder brought by the operating system.

Supported File Formats

  • AAC
  • AC3
  • AIFF
  • ALAC
  • ENC
  • FLAC
  • FX
  • MIDI
  • MIX
  • MP3
  • MP2
  • MP1
  • MPC
  • OGG
  • SPX
  • WAV
  • WV
The version of NativeBASS in this plugin uses the operating system's mp3 decoder. If you experience problems with mp3 playback you can add BASS' native mp3 support (see below).

System Requirements

For mp3 playback you need an installed mp3 decoder for your operating system. For Windows an installed Windows Media Player should be sufficient.

Adding mp3 support

If you have problems with mp3 playback or if you do not want to depend on an installed mp3 decoder, you can download a version of NativeBASS that brings built-in mp3 support. Under http://jerome.jouvie.free.fr/nativebass/index.php you can download an archive containing the binaries and the source code for NativeBASS. Download the archive and copy the contents of the contained lib folder to <smartsound>/plugins/nativebass/NativeBass where <smartsound> is your smartsound home folder.
