IrrKlang Plugin

Warning: The IrrKlang Plugin is a prototype which contains a number of bugs. We recommend the usage of the NativeBASS plugin

Supported File Formats

  • RIFF WAVE (*.wav) (with bugs, sounds cannot be repeated)
  • Ogg Vorbis (*.ogg)
  • Free Lossless Audio Codec (*.flac)
  • Amiga Modules (*.mod)
  • Impulse Tracker (*.it)
  • Scream Tracker 3 (*.s3d)
  • Fast Tracker 2 (*.xm)
Support for mp3 can be added via a plugin of irrKlang, as described below.

System Requirements

Windows

  • Java 7 32 Bit

Linux

  • Java 7
  • Installed package libasound2-dev

Adding mp3 support

For legal reasons we are not allowed to deliver mp3 support with this plugin. However, it can still be added. You can download the full version of IrrKlang at http://www.ambiera.com/irrklang/downloads.html. Download version 1.3.0b and copy the folder in the zip file to <smartsound>/plugins/irrklang where <smartsound> is your smartSound home folder. Nevertheless, mp3 playback sometimes leads to disturbing, cracking sounds.
